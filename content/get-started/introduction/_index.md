## 关于本系列

在本系列指南中，你将通过实践全面上手 Docker：首先在本地安装并配置 Docker Desktop；
随后运行你的第一个容器，理解容器化的基本原理与优势；再学习构建你的第一个 Docker 镜像，
掌握创建高效、可复用镜像的方法。最后，你将把镜像发布到 Docker Hub，
与更广泛的社区共享你的成果，并充分利用 Docker 强大的生态进行协作式开发与部署。

## 你将学到什么

- 安装并配置 Docker Desktop
- 运行你的第一个容器
- 构建你的第一个镜像
- 将你的镜像发布到 Docker Hub
