> [!WARNING]
>
> 删除仓库会删除其中的所有镜像及其构建设置。此操作不可撤销。

1. 登录 [Docker Hub](https://hub.docker.com)。
2. 选择 **My Hub** > **Repositories**。

   将显示你的仓库列表。

3. 选择一个仓库。

   将进入该仓库的 **General** 页面。

4. 选择 **Settings** 选项卡。
5. 选择 **Delete repository**。
6. 输入仓库名称以确认。
7. 选择 **Delete Repository Forever**。
