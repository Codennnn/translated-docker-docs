Docker Offload 是一项完全托管的服务,让你能够使用熟悉的 Docker 工具将容器的构建和运行任务卸载到云端。
它提供云基础设施来支持快速、稳定的构建,以及运行大语言模型(LLM)或机器学习流水线等计算密集型工作负载。

在下面的章节中,你将了解 Docker Offload 的相关知识、如何设置它、
如何在工作流中使用它,以及如何排查常见问题。
