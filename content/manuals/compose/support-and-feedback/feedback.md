你可以通过多种方式为 Docker Compose 提供反馈。

### 在 GitHub 报告缺陷或问题

如需报告缺陷或问题，请访问 [Docker Compose on GitHub](https://github.com/docker/compose/issues)

### 通过社区 Slack 渠道反馈

你也可以通过 `#docker-compose` [Docker Community Slack](https://dockr.ly/comm-slack) 频道反馈。
