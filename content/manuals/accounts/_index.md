本节介绍个人 Docker 账户与 Docker ID。
不涵盖组织、公司或管理员角色。

你需要 Docker 账户才能：
- 创建 Docker ID
- 访问 Docker 的产品与服务，如 Docker Hub 与 Docker Desktop
- 接收组织邀请
- 管理个人设置与安全功能
